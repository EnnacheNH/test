Le Vieux Sage

Un bot discord afin de demander des informations lors de jeux de rôles. Développé avec Python et des API.

Un script pour les gouverner tous

Afin de faciliter mes parties de jeu de rôles, j'ai créé un bot discord. Les joueurs peuvent alors lui demander des informations sur des sortilèges et les obtenir rapidement. Cependant, en plus de cela, les joueurs peuvent facilement changer les informations que renvoie le bot sans avoir à toucher au code.

J'ai donc un script python qui me permet de mettre en lien ces éléments. Il utilise les éléments suivants:

  • L'API discord
  • Dropbox avec request
  • Un fichier xlsx

J'ai choisi d'utiliser Dropbox plutôt que Google Sheet pour héberger la base de données des joueurs par soucis de la propriété des données. Cependant j'ai également créé une autre version du projet, plus pratique pour les utilisateurs avec les éléments suivants:
  • L'API discord
  • L'API Google Sheet
  • Une base de données SQL

Sewing Drawer

Une application android destinée à faire un inventaire de couture. Développé avec Python et la librairie Kivy.

Un Projet python ?

Afin de pouvoir créer des applications plus poussées, je me suis penché vers le language Python. Après quelques projets d'autoformation, voici Sewing Drawer, mon premier projet python terminé.
J'ai créé toute l'interface graphique de l'application avec Kivy, et j'ai utilisé SQLite pour gérer les bases de données.

Ainsi l'application permet de faire des inventaires de coutures, avec des catégories différentes:

  • Le tissu
  • Les projets
  • Une liste de courses
  • Les mesures
Par contre, le point le plus important du projet est que j'ai ainsi pu créer un fichier APK de l'application afin d'avoir son inventaire directement sur son portable. Vous pouvez d'ailleurs trouver ce fichier ici.

Spidy Vs The Bugs

Un jeu vidéo platformer 2D fait en un mois lors d'une Game Jam. Fait avec Unity en C#.

Une gamejam !

J'ai créé ce jeu lors d'une gamejam, la Game-Off 2021 de itch.io dont le thème était "BUG". Ainsi, le jeu aborde le thème de deux manières : avec des insectes (traduction de bug) et avec des papillons corrompus (et donc buggés). Cette gamejam était une expérience intéressante et elle est venu avec son lot de défis :

  • Le respect du thème de la gamejam
  • Le respect de la deadline
  • J'ai pu avoir le retour d'autres participants de la gamejam sur mon jeu, mais j'ai également pu tester leur jeu. Cela m'a permis de voir différentes approches du thème et ce que j'aurais pu modifier sur mon jeu

Vous pouvez retrouver le jeu sur itch.io.

Dagger Dash

Un jeu vidéo platformer 2D dont le principe est de lancer des dagues et se téléporter. Fait avec Unity en C#.

Le premier projet

Dévelopé avec Unity, c'est le tout premier projet de programmation que j'ai fait (hormis les projets Arduino et les divers projets d'autoapprentissage). En effet, étant passionné de jeu vidéo, c'était l'excuse parfaite pour apprendre à coder ! Pour un premier projet, ce fût assez difficile mais je suis assez fière du résultat. Voici une liste non exhaustive des différentes difficultés que j’ai dû surmonter :

  • Déboguer encore et encore pour avoir une application fonctionnelle
  • Gérer les interactions entre les différents éléments (joueur, monde, ennemi, dague)
  • Mettre en place un système de mémoire pour sauvegarder la progression du joueur
  • Tester afin d’avoir un gameplay agréable
  • Mettre en place une interface graphique

Vous pouvez retrouver le jeu sur itch.io.

Des applications concrètes
