Sewing Drawer

Un Projet python ?

Afin de pouvoir créer des applications plus pousser, je me suis penché vers le language Python. Après quelques projets d'autoformation, voici Sewing Drawer, mon premier projet python terminé.
En soi le plus difficile lors de ce projet n'a pas été le language Python mais plutôt la librairie Kivy.
En effet j'ai créé toute l'interface graphique de l'application avec Kivy, et j'ai utilisé sqlite pour gérer les bases de données.

Ainsi l'application permet de faire des inventaires de coutures, avec des catégories différentes:

  • Le tissu
  • Les projets
  • Une liste de courses
  • Les mesures
Chaque catégorie possède ses propres attributs et cela permet également de trier les objets selon ces attributs.

Par contre, le point le plus important du projet est que j'ai ainsi pu créer un fichier APK de l'application afin d'avoir son inventaire directement sur son portable.
